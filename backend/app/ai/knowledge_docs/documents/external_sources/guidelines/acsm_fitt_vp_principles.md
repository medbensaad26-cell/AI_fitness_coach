# ACSM's FITT-VP Principles for Exercise Prescription

## Overview
The FITT-VP framework ensures exercise programs are safe, effective, and individualized. It stands for Frequency, Intensity, Time, Type, Volume, and Progression.

## Components
- **Frequency (F)** : How often an activity is performed.
- **Intensity (I)** : How hard the activity is.
- **Time (T)** : Duration of each session.
- **Type (T)** : Mode of exercise (aerobic, resistance, flexibility, neuromotor).
- **Volume (V)** : Total amount of work (e.g., sets × reps × load, or MET‑minutes).
- **Progression (P)** : How the program advances over time.

## Aerobic Exercise Recommendations
| FITT Component | Recommendation |
|----------------|----------------|
| Frequency | ≥3–5 days/week |
| Intensity | Moderate (40–59% HRR/VO₂R) and/or vigorous (60–89% HRR) |
| Time | 30–60 min/day of moderate, or 20–30 min/day of vigorous |
| Type | Rhythmic, large‑muscle activities (walking, cycling, swimming) |
| Volume | ≥500–1000 MET‑min/week |
| Progression | Increase duration 5–10 min every 1–2 weeks, then intensity |

## Resistance Exercise Recommendations
| Component | Recommendation |
|-----------|----------------|
| Frequency | ≥2 non‑consecutive days/week per muscle group |
| Intensity | 60–80% 1RM for strength; 40–60% for beginners |
| Sets/Reps | 2–4 sets of 8–15 reps for general fitness |
| Type | Multi‑joint and single‑joint exercises, balance of push/pull |
| Progression | Increase load when current reps can be completed with proper form |

## Flexibility Recommendations
- Frequency: ≥2–3 days/week (daily is ideal).
- Stretch to the point of mild discomfort, hold 10–30 s, repeat 2–4 times.
- Static, dynamic, and PNF stretching all effective.

## Safety & Individualization
Always tailor to the individual's health status, goals, and experience. Pre‑participation screening (e.g., PAR‑Q+) is essential. Progress gradually to avoid injury.
